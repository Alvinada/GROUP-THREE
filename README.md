# SAFE HAVEN RIGHTS ORGANISATION 🛡️

## Project Overview
This website addresses the lack of awareness and access to **Human Rights information** among young parents. Our platform directs users on how to access:
*   **Health care**
*   **Counseling & Mental Health support**
*   **Education**

---

## 🛠️ Technical Implementation
The project is built using **HTML5** and managed via **GitHub** for version control.
*   **Semantic Navigation**: Used `<ul>` (unordered lists) and `<li>` (list items) to create a structured menu.
*   **Hypertext References**: Used `<a href>` tags to create clickable links for seamless navigation between pages.
*   **Collaborative Workflow**: Managed through GitHub branches and invitations.
